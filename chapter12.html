#️⃣ Hashfunctie

Definitie: Mapt strings naar getallen (sleutels naar array-indexen)

Vereisten:

  • Consistent: dezelfde invoer → dezelfde uitvoer
  • Mapt verschillende invoer naar verschillende getallen
  • Kent de arraygrootte en geeft geldige indexen terug

️ Hashtabel

Formule: Hashfunctie + array = hashtabel

Mapt sleutels naar waarden via een hashfunctie om de index te berekenen

Ook bekend als:

Hashmaps, dictionaries, associatieve arrays

Snelheid: O(1) gemiddeld - onmiddellijke lookup!

Complexiteitsvergelijking

  • Array/linked list: O(n) zoeken
  • Gesorteerde array: O(log n) zoeken
  • Hashtabel: O(1) zoeken ✓

Botsingen

Probleem: Twee sleutels mappen naar dezelfde index

Verjaardagsparadox:

Bij 23 mensen: 50% kans op dezelfde verjaardag
Bij 60 mensen: 99% kans!

Botsingen zijn waarschijnlijker dan je denkt!

Load factor

Formule: items / grootte

Regel: Resize wanneer load factor > 0,7

Tabelgrootte:

  • 1,3 × max sleutels
  • Moet een priemgetal zijn

Belangrijkste inzichten

  • Hashtabel = O(1)-lookups
  • Hashfunctie mapt sleutels naar indexen
  • Botsingen gebeuren vaak
  • Load factor > 0,7 → resize
  • Praktijk: contacten, logins, bestandssystemen
