#️⃣ Hash Function

Definition: Maps strings to numbers (keys to array indices)

Requirements:

  • Consistent: same input → same output
  • Maps different inputs to different numbers
  • Knows array size, returns valid indices

️ Hash Table

Formula: Hash Function + Array = Hash Table

Maps keys to values using hash function to compute index

Also known as:

Hash maps, dictionaries, associative arrays

Speed: O(1) average - instant lookup!

Complexity Comparison

  • Array/Linked List: O(n) search
  • Sorted Array: O(log n) search
  • Hash Table: O(1) search ✓

Collisions

Problem: Two keys map to same index

Birthday Paradox:

With 23 people: 50% chance of same birthday
With 60 people: 99% chance!

Collisions are MORE likely than you think!

Load Factor

Formula: items / size

Rule: Resize when load factor > 0.7

Table Size:

  • 1.3 × max keys
  • Should be prime number

Key Takeaways

  • Hash table = O(1) lookups
  • Hash function maps keys to indices
  • Collisions happen often
  • Load factor > 0.7 → resize
  • Real-life: contacts, logins, file systems
