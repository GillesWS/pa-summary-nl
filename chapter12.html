#️⃣ Hash Functie

Definitie: Maps strings naar numbers (keys naar array indices)

Requirements:

  • Consistent: dezelfde input → dezelfde output
  • Maps different inputs naar different numbers
  • Knows array grootte, returns valid indices

️ Hash Tabel

Formule: Hash Functie + Array = Hash Tabel

Maps keys naar waarden gebruik van hash functie naar compute index

Also known als:

Hash maps, dictionaries, associative arrays

Speed: O(1) gemiddelde - instant lookup!

Complexiteit Vergelijking

  • Array/Linked Lijst: O(n) zoeken
  • Gesorteerd Array: O(log n) zoeken
  • Hash Tabel: O(1) zoeken ✓

Collisions

Probleem: Two keys map naar dezelfde index

Birthday Paradox:

Met 23 people: 50% chance van dezelfde birthday
Met 60 people: 99% chance!

Collisions zijn MORE likely dan you think!

Load Factor

Formule: items / grootte

Rule: Resize when load factor > 0.7

Tabel Grootte:

  • 1.3 × max keys
  • Should be prime number

Belangrijkste inzichten

  • Hash tabel = O(1) lookups
  • Hash functie maps keys naar indices
  • Collisions happen often
  • Load factor > 0.7 → resize
  • Real-life: contacts, logins, file systems
